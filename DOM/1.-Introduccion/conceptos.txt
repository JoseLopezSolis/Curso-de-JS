Nodo - Es cualquier etiqueta del cuerpo del HTML

TIPOS DE NODOS QUE EXISTEN EN DOOM

>Document: El nodo document es el nodo raiz, a partir de cual derivan el resto de nodos
>Element: Nodos definidos por etiquetas HTML (Como etiqueta p, h1, etc)
>Text: El texto dentro de un nodo element se considera un nuevo nodo hijo de tipo text (texto)
>Attribute: Los atributos de las etiquetas definen nodos, (en Javascript no lo veremos como nodos sino como informacion asociada al nodo de tipo element.)
>Comentarios y otros: Los comentarios y otros elementos como las declaraciones doctype en cabeceras
